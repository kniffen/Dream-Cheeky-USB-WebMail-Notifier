# Dream-Cheeky-USB-WebMail-Notifier

![Product promo](https://i.imgur.com/5Vropmk.jpg)



## Usage
```JS
const usbwn = require('./usbwn.js')

// Set colors
// Availiable presets are pink, red, orange, yellow, green, blue, purple
usbwn.setColor('red')

// Set RGB values
usbwn.setRGB(200, 6, 35)

// Turn off the LED
usbwn.off()
```
